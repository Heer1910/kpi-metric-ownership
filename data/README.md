# Data Files (Not Included in Repository)

This directory should contain the Instacart dataset CSV files:

- `orders.csv`
- `order_products__prior.csv`
- `order_products__train.csv`
- `products.csv`
- `aisles.csv`
- `departments.csv`

## Download Instructions

Download the dataset from Kaggle:
[Instacart Market Basket Analysis](https://www.kaggle.com/datasets/psparks/instacart-market-basket-analysis)

Extract all CSV files to this directory before running the analysis.

## Why Not Included?

These files are excluded from version control because:
- Total size: ~700 MB (too large for GitHub)
- Publicly available dataset (easy to download)
- Data licensing considerations

## File Sizes Reference

- `orders.csv`: ~109 MB
- `order_products__prior.csv`: ~578 MB
- `order_products__train.csv`: ~25 MB
- `products.csv`: ~2.2 MB
- `aisles.csv`: ~3 KB
- `departments.csv`: ~270 bytes
